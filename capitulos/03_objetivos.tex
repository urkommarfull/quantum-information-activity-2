\chapter{Objetivos}

\section{Objetivos generales}
\begin{itemize}
    \item Describir los principales protocolos de intercambio de claves cuánticas de variable discreta.
    \item Comparar sus fundamentos, requisitos tecnológicos, puntos fuertes y debilidades.
    \item Relacionar cada protocolo con escenarios prácticos de uso y con sus limitaciones de seguridad.
\end{itemize}

\section{Objetivos específicos}
\begin{itemize}
    \item Definir un marco uniforme de análisis aplicable a todos los protocolos estudiados.
    \item Explicar el flujo operativo de cada protocolo mediante ejemplos originales y comprensibles.
    \item Identificar las amenazas más relevantes: espionaje interceptar-reenviar, pérdidas, ataques de división de número de fotones, ataques a detectores y supuestos de confianza.
    \item Elaborar una tabla comparativa que facilite el posicionamiento de cada protocolo.
    \item Formular conclusiones útiles para seleccionar un protocolo en un caso de despliegue realista.
\end{itemize}
